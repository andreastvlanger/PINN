
# An Implementation of AD-PINNs and FD-PINNs

## Overview

This repository contains an implementation of physics-informed neural networks based on automatic differentiation (AD) and finite differences (FD) and contains the numerical experiments from the paper 

[1] Andreas Langer. The ill-posed foundations of physics-informed neural networks and their finite difference variants. arXiv preprint arXiv:2601.07017, 2026. [link](https://arxiv.org/abs/2601.07017)
                                             
Read and use at your own risk.

If you use the code, please make sure to cite [1].
