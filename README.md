
# An Implementation of AD-PINNs and FD-PINNs

## Overview

This code is based on the paper 

[1] Andreas Langer. 'The ill-posed nature of physics-informed neural networks and their finite difference variants'. arXiv:2409.05569, 2024.
                                             
Read and use at your own risk.

This repository contains an implementation of physics-informed neural networks based on automatic differentiation (AD) and finite differences (FD).

If you use the code, please make sure to cite [1]. 

For more information consider [1].
